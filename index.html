Instruction to play

The objective of the game is to find all the matching pairs. Test your memory with this memory game. The game starts with an equal number of cards, all the cards will be face down. The test is to find all the matching cards. Start the game by selecting a card, the selected card will flip over revealing an image. Then try to find another card that has the same image as the first. If the second card selected does not match the first, then both cards will be flipped back face down. Try to remember which image was in which spot. It requires observation, concentration and a good memory to win! To start the game click 'Play Now!'
